# Expense Tracker

A simple Python project to track daily expenses. Users can add expenses, view all records, calculate total spending, and store data in a CSV file.

## Features
- Add expenses
- View saved expenses
- Calculate total spending


## How to Run
1. Clone the repository  
   ```bash
   git clone https://github.com/your-username/expense-tracker.git
   ```
2. Navigate to the folder  
   ```bash
   cd expense-tracker
   ```
3. Run the script  
   ```bash
   python3 expense_tracker.py
   ```
